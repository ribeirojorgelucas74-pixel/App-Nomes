# Desenvolvimento do App

Este sistema automatiza a gestão. Uma aplicação web progressiva projetada para facilitar a organização, registo e gestão de bases de dados personalizáveis. Com uma interface moderna e intuitiva, o aplicativo permite que utilizadores gerenciem desde turmas de alunos até cadastros de pacientes ou produtos com eficiência.

## 🚀 Funcionalidades Principal

- **Criação de Bases de Dados Personalizadas:** Defina quais campos (CPF, Email, Matrícula, Status, etc.) cada base deve conter
- **Formulários Dinâmicos:** O sistema adapta automaticamente os campos de entrada com base na configuração da base de dados selecionada
- **Interface Dark Mode:** Design otimizado com suporte a tema escuro para melhor conforto visual

# 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3 (Syne & DM Sans Fonts)
- **Lógica:** JavaScript (Vanilla JS) para manipulação de DOM e lógica de estado 
- **Deploy:** GitHub Pages
